# Unity Gameplay Programming Samples  
Ervin Setyawan  

This repository contains selected Unity gameplay programming prototypes focused on system implementation and technical structure.
The purpose of these samples is to demonstrate gameplay logic, modular architecture, and multiplayer fundamentals rather than visual polish.

## Technical Focus

- Deterministic ECS-based multiplayer simulation  
- Matchmaking & Quick Play system  
- PlayFab integration (account & avatar sync)  
- Input synchronization (movement & jump)  
- Separation between simulation and presentation layer  
- Modular gameplay system design
